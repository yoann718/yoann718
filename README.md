## 🚀 À propos de moi

Je suis en pleine montée en compétences dans le domaine des **systèmes et réseaux**, avec une vraie envie de comprendre comment fonctionne une infrastructure de bout en bout.

Ce qui m’intéresse particulièrement, c’est de partir d’une idée ou d’un besoin technique, puis de construire une solution complète : serveurs, réseau, services, sécurité, supervision, documentation et automatisation.

J’aime tester, casser, corriger, recommencer et surtout comprendre **pourquoi ça fonctionne** ou **pourquoi ça ne fonctionne pas**.
